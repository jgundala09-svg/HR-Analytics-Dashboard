# HR Analytics Dashboard

## Overview

This project is an interactive HR Analytics Dashboard developed using Microsoft Power BI.

The dashboard helps analyze employee data, attrition trends, salary distribution, department performance, gender distribution, experience, and employee demographics.

## Dashboard Preview

![Dashboard](images/dashboard.png)

---

## Features

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Experience
- Department-wise Employee Count
- Salary Slab Analysis
- Job Role & Satisfaction Analysis
- Gender Distribution
- Experience Trend
- Age Group Distribution

---

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Excel / CSV

---

## Dataset

HR Employee Dataset

---

## Key Insights

- Administration department has the highest employee count.
- Most employees belong to the 26–35 age group.
- Attrition rate is 16.3%.
- Average employee age is 36.94 years.
- Average experience is 7.04 years.
