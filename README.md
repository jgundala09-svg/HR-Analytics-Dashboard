# HR Analytics Dashboard

 About the Project

The HR Analytics Dashboard is an interactive Business Intelligence solution developed using **Microsoft Power BI**. This project transforms raw HR data into meaningful insights through dynamic dashboards and visual reports. It helps organizations monitor employee performance, analyze attrition trends, evaluate department-wise workforce distribution, and understand salary, age, gender, and experience patterns. The dashboard enables HR professionals to make informed, data-driven decisions with the help of interactive KPIs, charts, and filters.


##  Project Objectives

- Analyze employee workforce data
- Monitor employee attrition and retention
- Compare salary distribution across employees
- Evaluate department-wise employee performance
- Understand employee demographics and experience
- Support HR decision-making with interactive dashboards


## Dashboard Highlights

-  Total Employees
-  Active Employees
- Attrition Count & Attrition Rate
- Salary Slab Analysis
-  Department-wise Employee Distribution
-  Job Role & Job Satisfaction Analysis
-  Gender Distribution
-  Age Group Analysis
- Experience Trend Analysis
- Interactive Filters & KPIs


##  Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV


##  Dataset

The dashboard is built using an HR employee dataset containing employee information such as department, salary, age, gender, job role, years of experience, and attrition status.


##  Business Insights

- Analyzed employee distribution across departments.
- Identified attrition trends and retention patterns.
- Compared employee salaries across different salary slabs.
- Evaluated job satisfaction among various job roles.
- Analyzed workforce demographics including age, gender, and experience.
